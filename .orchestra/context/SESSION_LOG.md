# Session Log

> **Purpose:** Record of what happened in each work session. Newest first.
> **Update frequency:** At session close.
> **Owner:** Gemini writes entries. Claude may write if Gemini is unavailable.
> **Rule:** Max 20 lines per entry. Be factual, not narrative. If an entry is getting long, you're including too much detail — put the detail in history/ files instead.
