# Decisions

> **Purpose:** Durable decisions with full traceability. This is the source of truth for "why did we do it this way?"
> **Update frequency:** Whenever a decision is made. Gemini extracts these at session close.
> **Owner:** Gemini records. Claude and Human make decisions.
> **Rule:** Never edit a decision after recording. If a decision is reversed, add a new entry that references the old one.
